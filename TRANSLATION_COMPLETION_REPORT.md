## Translation Methodology

### Keyword Localization (Not Literal Translation)

The translation prioritizes how Russian-speaking users actually search and communicate:

| English Concept | Russian Translation | Why This Choice |
|-----------------|-------------------|-----------------|
| Transcript | Транскрипция / Субтитры | Users search for "субтитры" (subtitles), but formal product copy uses "транскрипция" |
| Download | Скачать | Standard Russian UX verb for file downloads |
| Generator | Генератор | Standard SaaS terminology in Russian tech |
| Video Script | Сценарий | Professional term for video content scripts |
| Viral Hooks | Вирусные зацепки | "Зацепки" is colloquial but high-impact in Russian creator culture |
| Bulk Import | Массовый импорт | Standard business/enterprise Russian |
| Cloud Storage | Облачное хранилище | Standard tech term (literal but appropriate) |
| Watermark | Водяной знак | Standard in Russian UX/software |
| Cover Image | Обложка | Russian users call video thumbnails "обложка" |
| Viral/Virality | Вирусный / Вирусность | Context-dependent; "вирусный" for adjectives, "вирусность" for abstract noun |
