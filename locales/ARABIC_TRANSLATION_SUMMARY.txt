TRANSLATION APPROACH
================================================================================

Philosophy: Keyword Localization (not literal translation)
- Native Arabic speakers search for "نص الفيديو" not "ترانسكريبت"
- "استيراد جماعي" is established in Arabic e-commerce
- "التخزين السحابي" widely understood post-cloud adoption

Key Terms Localized:
  Transcript → نص الفيديو (video text)
  Download → تحميل / تنزيل
  Generator → مولّد (active participle)
  Script → سيناريو (media industry standard)
  Hooks → خطافات (proven hooks)
  Viral → فيروسي (or منتشر as alternative)
  Bulk import → استيراد جماعي
  Cloud storage → التخزين السحابي
  Watermark → علامة مائية
  Cover image → صورة الغلاف
