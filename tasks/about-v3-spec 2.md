# About V3 — Shared Class Name Spec

## Wrapper
- `.about-v3` — main wrapper on `<main>`, replaces `.about-us-page`

## Section 1: Hero
- `.av3-hero`
- `.av3-hero__badge` — small pill label
- `.av3-hero__heading` — large gradient text
- `.av3-hero__subtitle` — muted description
- `.av3-hero__cta` — button wrapper
- `.av3-hero__cta-btn` — the pill button itself
- `.av3-hero__social` — social proof row (avatars + text)
- `.av3-hero__avatars` — avatar stack
- `.av3-hero__glow` — background radial glow effect

## Section 2: Stats
- `.av3-stats`
- `.av3-stats__card` — the rounded container holding all stats
- `.av3-stats__grid` — flex row of stat items
- `.av3-stats__item` — individual stat
- `.av3-stats__number` — large cyan number
- `.av3-stats__label` — muted label text

## Divider
- `.av3-divider` — full-width gradient line (cyan → pink)

## Section 3: What Is TokScript
- `.av3-what`
- `.av3-badge` — reusable section badge (pill with icon + text)
- `.av3-what__grid` — two-column layout
- `.av3-what__content` — left text column
- `.av3-what__heading` — section heading
- `.av3-what__text` — body text paragraph
- `.av3-what__mockup` — right column with faux UI

## Section 4: Problem
- `.av3-problem`
- `.av3-problem__columns` — before/after columns
- `.av3-problem__col` — each column
- `.av3-problem__col-title` — "Before" / "After" label
- `.av3-problem__card` — individual card
- `.av3-problem__card--before` — red/muted tint
- `.av3-problem__card--after` — cyan/green tint
- `.av3-problem__card-icon` — icon area
- `.av3-problem__card-title` — card title
- `.av3-problem__card-desc` — card description

## Section 5: Users
- `.av3-users`
- `.av3-users__grid` — 5-card grid (3+2 on desktop)
- `.av3-users__card` — user persona card
- `.av3-users__card-icon` — icon circle
- `.av3-users__card-title` — persona title
- `.av3-users__card-desc` — short description

## Section 6: Features
- `.av3-features`
- `.av3-feature-block` — large rounded container for each feature
- `.av3-feature-block--reverse` — reversed layout (mockup left, text right)
- `.av3-feature-block__inner` — inner flex container
- `.av3-feature-block__content` — text side
- `.av3-feature-block__label` — section label badge
- `.av3-feature-block__heading` — feature heading
- `.av3-feature-block__text` — description text
- `.av3-feature-block__mockup` — mockup side
- `.av3-feature-block__sub-grid` — 3-column sub-features
- `.av3-feature-block__sub-item` — individual sub-feature
- `.av3-feature-block__sub-title` — sub-feature title
- `.av3-feature-block__sub-desc` — sub-feature description
- `.av3-feature-block__divider` — gradient line at bottom of feature block

## Mockup Components (CSS-only faux UI)
- `.av3-mockup` — mockup container
- `.av3-mockup__toolbar` — fake toolbar with dots
- `.av3-mockup__dot` — red/yellow/green dot
- `.av3-mockup__body` — content area
- `.av3-mockup__row` — fake content row
- `.av3-mockup__label` — fake label text
- `.av3-mockup__btn` — fake button

## Section 7: How It Works
- `.av3-how`
- `.av3-how__grid` — 6-column grid (or 3x2)
- `.av3-how__step` — step card
- `.av3-how__step-num` — large step number
- `.av3-how__step-title` — step title
- `.av3-how__step-desc` — step description
- `.av3-how__step-arrow` — arrow connector between steps

## Section 8: Results
- `.av3-results`
- `.av3-results__card` — table container card
- `.av3-results__table` — the table element
- `.av3-results__table th` — header cells
- `.av3-results__table td` — data cells
- `.av3-results__highlight` — highlighted "After" values

## Section 9: Why Different
- `.av3-why`
- `.av3-why__grid` — 3x2 grid
- `.av3-why__card` — differentiation card
- `.av3-why__card-title` — card title
- `.av3-why__card-desc` — card description

## Section 10: Ecosystem
- `.av3-ecosystem`
- `.av3-ecosystem__grid` — 4-column grid
- `.av3-ecosystem__card` — product card
- `.av3-ecosystem__card-name` — product name
- `.av3-ecosystem__card-desc` — product description

## Section 11: FAQ
- `.av3-faq`
- Uses Bootstrap Accordion classes (`.faq-section` existing pattern)

## Section 12: Final CTA
- `.av3-final-cta`
- `.av3-cta-card` — big gradient card
- `.av3-cta-card__heading` — heading
- `.av3-cta-card__text` — subtitle
- `.av3-cta-card__btn` — CTA button
- `.av3-cta-card__note` — "No credit card required" text
