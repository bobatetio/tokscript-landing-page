# XML Sitemap | TokScript

**URL:** https://tokscript.com/sitemap.xml

**File Source:** src/app/sitemap.js

## Overview

This is the XML sitemap file generated by Next.js at build/runtime. It lists all publicly-accessible URLs on the TokScript site with metadata for search engines.

**Format:** XML (application/xml)

**Purpose:** Help search engines discover and crawl all pages on the site efficiently

## Sitemap Generation

**Function:** Default export from sitemap.js

**Base URL:** https://tokscript.com

**Total Entries:** 37 URLs

**Entry Structure:** Each URL includes:
- url: Full URL path
- lastModified: Date modified (set to current date on generation)
- changeFrequency: How often content changes (weekly, monthly, yearly)
- priority: Crawl priority relative to other URLs (0.5 to 1.0)

## Pages NOT in Sitemap

The following pages are intentionally excluded from the sitemap:

- **/upgrade** - Internal upgrade/checkout page (user-specific)
- **/cancel-demo** - Internal demo/testing page
- **/not-found** - Dynamic 404 error page (caught by wildcard routing)
- **Any [catch-all] routes** - Non-existent paths

## Change Frequency Strategy

- **Weekly:** Homepage, language variants, platform-specific tools, main features (content/pricing updated frequently)
- **Monthly:** Advanced features (extension, cloud storage, team), information pages, API/integration pages
- **Yearly:** Legal pages (privacy, terms, legal disclaimer - rarely change)

## Impact on SEO

1. **Crawl Efficiency:** Search engines prioritize high-priority URLs
2. **Discovery:** New pages are discoverable via sitemap
3. **Hreflang Support:** Language variants help Google understand localization
4. **Freshness Signals:** Change frequency indicates content volatility
5. **Indexation:** Ensures all public URLs are indexed in search results

## Technical Details

**Generation Method:** Next.js dynamic sitemap.js export

**Update Frequency:** Sitemap is regenerated at build time or request time (depending on deployment)

**Search Engine Submission:** Should be submitted to Google Search Console and Bing Webmaster Tools at https://tokscript.com/sitemap.xml
